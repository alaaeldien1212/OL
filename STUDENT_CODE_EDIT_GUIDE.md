# 📖 دليل تعديل رموز الطلاب - Student Code Edit Guide

## 🎯 نظرة عامة - Overview

هذه الميزة تتيح للمعلمين تعديل رموز الوصول الخاصة بالطلاب بسهولة وأمان.

This feature allows teachers to edit student access codes easily and securely.

---

## 📍 الوصول للميزة - Accessing the Feature

### 1️⃣ تسجيل الدخول كمعلم
Login as a teacher using your access code

### 2️⃣ الانتقال لإدارة الطلاب
Navigate to: **لوحة التحكم → إدارة الطلاب** (Dashboard → Student Management)

URL: `/teacher/students`

---

## ⚡ كيفية الاستخدام - How to Use

### 🔍 عرض رمز الطالب - View Student Code

في بطاقة الطالب، ستجد قسم "رمز الوصول":
```
┌─────────────────────────────────┐
│ رمز الوصول                      │
│ ┌─────────────────────────────┐ │
│ │ ABC12345          ✏️  📋   │ │
│ └─────────────────────────────┘ │
└─────────────────────────────────┘
```

### ✏️ تعديل الرمز - Edit the Code

**الخطوة 1:** اضغط على زر التعديل ✏️
```
┌─────────────────────────────────┐
│ رمز الوصول                      │
│ ┌─────────────────────────────┐ │
│ │ [ABC12345____________]      │ │  ← حقل الإدخال
│ ├─────────────┬───────────────┤ │
│ │    حفظ      │     إلغاء     │ │  ← أزرار الحفظ والإلغاء
│ └─────────────┴───────────────┘ │
└─────────────────────────────────┘
```

**الخطوة 2:** أدخل الرمز الجديد
- يتم التحويل تلقائياً للأحرف الكبيرة
- أحرف إنجليزية وأرقام فقط
- 4 أحرف على الأقل

**الخطوة 3:** اضغط "حفظ" لحفظ التغييرات أو "إلغاء" للتراجع

---

## ✅ قواعد الرموز - Code Rules

### ✔️ مسموح - Allowed
```
✓ ABC123      - أحرف وأرقام
✓ STUDENT1    - أحرف إنجليزية كبيرة
✓ 2024MATH    - أرقام وأحرف
✓ ABCD1234    - أي طول 4 أحرف أو أكثر
```

### ✖️ غير مسموح - Not Allowed
```
✗ abc         - أقل من 4 أحرف
✗ abc123      - أحرف صغيرة (سيتم تحويلها تلقائياً)
✗ ABC-123     - رموز خاصة (-, _, @, etc.)
✗ ABC 123     - مسافات
✗ أحمد123     - أحرف عربية
```

```
  → استخدام رموز خاصة أو أحرف غير مسموحة

✗ رمز الوصول يجب أن يكون 4 أحرف على الأقل
  → الرمز قصير جداً

✗ رمز الوصول مستخدم بالفعل
  → الرمز موجود لطالب أو معلم آخر
```

## 🛡️ الأمان - Security

### المعلم يمكنه تعديل رموز:
✅ الطلاب الذين أنشأهم فقط
✅ الطلاب في الصفوف المخصصة له

### المعلم لا يمكنه:
❌ تعديل رموز طلاب معلمين آخرين
❌ استخدام رموز مكررة
❌ استخدام رموز المعلمين أو المدراء

## 📊 أمثلة الاستخدام - Use Cases

### 🎯 حالة 1: تصحيح خطأ مطبعي
**الموقف:** الرمز الحالي `STDENT123` (خطأ في الكتابة)
**الحل:** تعديله إلى `STUDENT123`

### 🎯 حالة 2: رمز أسهل للتذكر
**الموقف:** الرمز العشوائي `X7KP9MQ2`
**الحل:** تغييره إلى `AHMED2024`

### 🎯 حالة 3: تنظيم الرموز
**الموقف:** رموز عشوائية مختلفة
**الحل:** استخدام نمط موحد مثل `CLASS3A01`, `CLASS3A02`, etc.

### 🎯 حالة 4: حل تعارض الرموز
**الموقف:** رمز مكرر مع طالب آخر
**الحل:** تعديل أحد الرموز لرمز فريد

## 🧪 اختبار الميزة - Testing the Feature

### اختبار سريع:
1. سجل دخول كمعلم
2. انتقل لإدارة الطلاب
3. اختر طالب
4. اضغط ✏️
5. أدخل `TEST123`
6. اضغط "حفظ"
7. تحقق من ظهور رسالة النجاح

### اختبار التحقق:
1. حاول إدخال `AB` (قصير جداً)
   → يجب أن يظهر خطأ
2. حاول إدخال `TEST-123` (رمز خاص)
   → يجب أن يظهر خطأ
3. حاول إدخال رمز موجود
   → يجب أن يظهر خطأ "مستخدم بالفعل"

## 🔍 استكشاف الأخطاء - Troubleshooting

### المشكلة: الزر لا يظهر
**الحل:** تأكد من تسجيل الدخول كمعلم وليس كطالب

### المشكلة: رسالة "Student not found"
**الحل:** الطالب ليس من طلابك، اتصل بالإدارة

### المشكلة: لا يتم الحفظ
**الحل:** تحقق من الاتصال بالإنترنت وصلاحيات الحساب

### المشكلة: رسالة "مستخدم بالفعل"
**الحل:** اختر رمز مختلف لم يستخدم من قبل

## 📝 ملاحظات مهمة - Important Notes

⚠️ **انتبه:**
- تغيير الرمز يؤثر فوراً على تسجيل دخول الطالب
- أخبر الطالب بالرمز الجديد
- الرمز القديم لن يعمل بعد التغيير
- جميع التغييرات مسجلة في سجل النظام

💡 **نصائح:**
- استخدم رموز سهلة التذكر للطلاب
- اتبع نمط موحد في الرموز
- احتفظ بقائمة بجميع الرموز
- لا تشارك الرموز مع غير المعنيين

---

## 🎓 أفضل الممارسات - Best Practices

### ✅ افعل - Do:
- استخدم رموز واضحة ومنظمة
- احفظ نسخة احتياطية من الرموز
- أخبر الطالب فوراً بالتغيير
- استخدم أنماط متسقة (مثل: GRADE3_01)

### ❌ لا تفعل - Don't:
- لا تستخدم رموز معقدة جداً
- لا تغير الرموز بدون إخبار الطالب
- لا تشارك رموز الطلاب علناً
- لا تستخدم رموز متشابهة جداً
